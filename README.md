# WDD 430 Web Full-stack Development course

In this repository you find the course structured by weeks. Every week has diferent assigments, practices or projects.

# Principal structure

## Repository Structure

- `project/` – Main projects for the course.
- `week01/` – Materials and exercises for week 1.
  - `learning/` – Learning exercises.
  - `practices/` – Practice tasks and mini-projects.
    - `todo-list/` – Example Todo List application.
- `week02/` – Materials and exercises for week 2.

Each folder contains its own configuration files and dependencies to keep projects independent.
